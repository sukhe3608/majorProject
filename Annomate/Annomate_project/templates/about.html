Virtual Canvas Project

About the Project

The Virtual Canvas Project is an innovative application that lets users draw using hand gestures. Built using OpenCV and MediaPipe, it tracks hand movements and converts them into actions like drawing, erasing, and color selection. This tool is ideal for artists, educators, and enthusiasts.

Shape Drawing

Draw various shapes like rectangles and circles with gestures. Toggle between filled or hollow shapes seamlessly.

Color Palette

Select different colors with hand gestures to make your artwork vibrant and expressive.

Gesture Control

Use simple hand gestures for actions like drawing, erasing, and even clearing the canvas.
